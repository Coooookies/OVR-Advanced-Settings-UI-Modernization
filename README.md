# [OVR Advanced Settings](https://github.com/OpenVR-Advanced-Settings/OpenVR-AdvancedSettings) UI Modernization

![Cover](cover.png)

English | [简体中文](README_CN.md)

Modernize your OVR Advanced Settings UI.

> [!WARNING]
> Some features are untested; the functionality of all options cannot be guaranteed.

## How to Use (Steam Version)

1. Find **OVR Advanced Settings** in your Steam Library.
2. Click the [Settings/Gear] icon, then select **[Manage] > [Browse local files]**.
3. In the opened folder, right-click the `res` folder, copy it, and paste it in the same directory to create a backup (e.g., rename it to `res_bak`).
4. In the folder containing this README, navigate to your desired language folder (e.g., `English`), and copy the `res` folder inside it to the OVR Advanced Settings root directory. Choose **[Replace the files in the destination]** if prompted.

## How to Use (GitHub Release Version)

1. Navigate to the OVR Advanced Settings installation folder.
2. Right-click the `res` folder, copy it, and paste it in the same directory to create a backup (e.g., rename it to `res_bak`).
3. In the folder containing this README, navigate to your desired language folder (e.g., `English`), and copy the `res` folder inside it to the OVR Advanced Settings root directory. Choose **[Replace the files in the destination]** if prompted.
